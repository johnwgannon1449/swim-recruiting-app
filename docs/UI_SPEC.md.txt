Lane4 — UI Specification
This document describes every screen, component, interaction, and display behavior in the Lane4 application as implemented in Lane4Advisor.jsx.

Global Layout
Container: min-height: 100vh, background #F8FAFC (near-white blue-grey), font-family: 'Figtree', sans-serif.
Max content width: 640px, centered with margin: 0 auto. All screens share this constraint.
Content padding: 1.5rem on left and right.
Global styles (injected via <style> tag in JSX):

Google Fonts loaded: Fraunces (weights 300, 600, 700, 800, including italic) and Figtree (weights 400, 500, 600, 700)
* { box-sizing: border-box; margin: 0; padding: 0; }
@keyframes fadeUp: from { opacity: 0; transform: translateY(14px) } to { opacity: 1; transform: translateY(0) } — used on all major view transitions
@keyframes spin: full rotation — used on the search loading spinner
Scrollbar: width: 3px, thumb color #CBD5E1, border-radius: 2px

Typography system:

Fraunces serif is used for all headings, school names, numbers, tier labels, and the "Ask →" button text.
Figtree sans-serif is used for all body text, labels, badges, vibe quotes, metadata, and UI controls.


Navigation Bar
Position: sticky, top: 0, z-index: 10. Always visible while scrolling.
Background: white, border-bottom: 1px solid #E2E8F0.
Left side: Lane4 logo (SVG: blue rect with three white bars, resembling a lane chart) + wordmark "Lane4" (Fraunces, weight 800, 1rem) + "· James" (Figtree, 0.78rem, #64748B slate).
Right side: Four tab buttons rendered as pill-shaped buttons.
Tab labels and logic:

"Explore" — maps to view "search"
"My List (N)" — maps to view "list". Shows (N) count when myList.length > 0; shows nothing when myList is empty.
"Reminders" or "Reminders 🔴" — maps to view "reminders". Shows 🔴 when urgentCount > 0 (any reminder is today or overdue).
"Profile" — maps to view "profile".

Active tab styling: border-color: #2563EB, background: #EFF6FF, color: #2563EB, font-weight: 600.
Inactive tab styling: border-color: #E2E8F0, background: white, color: #64748B.
All tabs: padding: 5px 14px, border-radius: 20px, border: 1.5px solid, font-size: 0.78rem, transition all 0.2s.

PDF Export Behavior
Triggered by: "⬇ Save PDF" button in Deep Dive.
Behavior:

window.open("", "_blank") opens a new browser tab
A self-contained HTML document is written to the new window via win.document.write()
The document auto-triggers window.print() via window.onload

PDF document structure:

Campus hero image (height: 220px, object-fit: cover, border-radius: 12px); hides on error
School name (h1, Fraunces 2rem)
Conference · Location (Figtree 0.85rem #64748B)
Admission badge + optional Hidden Ivy badge + optional STEM badge (inline-block with dynamic color backgrounds)
Swim snapshot block: "Swim Fit: {event} — {placeLabel} · ..." and "Program tier: {adjTier}"
All 8 deep dive sections: .section class (standard) or .bottom-line class (for Bottom Line)
Footer: "Generated by Lane4 · lane4.co" — Figtree 0.72rem #94A3B8, centered, margin-top: 2rem
Google Fonts loaded via @import in the style block
@media print { body { padding: 0; } } removes padding in print mode


Navigation / State Flow Summary
App loads
  → view = "search"
  → allSchoolResults computed (scoreAllSchools, deterministic)
  → vibe state initialized from JAMES.vibe

Explore tab (search):
  → Default: hero + search + suggested queries
  → User types or taps a query → runSearch() called
  → Loading spinner shown during API call
  → Results: answer banner + 3 result cards
    → "Deep Dive →" on any card → deepDive state set → DeepDive screen shown
      → "← Back" → clears deepDive → returns to results
      → "Save PDF" → new window opens → auto-prints
      → "Draft coach email" → email form shown → Copy button
    → "+ List" → adds to myList → button becomes "✓ Added"
    → "✕" → eliminates school → card disappears immediately
  → "← Ask something else" → clears results and query

My List tab:
  → Grouped school rows by status
  → Click any school → SchoolCRM panel shown
    → Change status → status updated live
    → Add/delete notes and reminders
    → "← Back" → returns to My List

Reminders tab:
  → All reminders from all schools, sorted by date
  → Read-only view (no editing here; edit from My List)

Profile tab:
  → Academic stats, swim times, vibe questions
  → Answered vibe: toggle/deselect live
  → Unanswered vibe: tap any option to answer
  → Changes immediately affect next deep dive prompt
